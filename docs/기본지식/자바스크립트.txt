자바 스크립트

자바 스크립트는 프로토 타입 기반의 객체 지향 언어이다.

객체란? 객체란 파이썬의 클래스와 같은 느낌 근데 이제 그 클래스에 변수와 함수가 같이 들어간다.

파이썬으로 치면 하나의.... 파이썬 파일?? 그니까 이제 고양이라는 객체에 고양이에 대한 정보와 함수가 들어감

프로토타입을 알기 전에 상속이라는 개념을 알아야한다. 
+
상속이란 이제.. 다른 객를 복제해서 재사용하는 것을 상속이라고 한다. 
음 아마도 파이썬에서 라이브러리를 불러오는 것처럼 하나의 객체를 만들 때 다른 객체에 있는 정보를 복제해서
가져온다. 모든 객체는 하나 이상의 객체에서 상속을 받기 시작한다.

근데 그러면.. 맨 처음에 객체를 만드는 건 뭐지..?

그리고 프로토타입이란 상속을 받은 객체를 프로토타입이라고 하네. 부모노드 느낌

Ex) var date = new Date();
여기서 date 라는 객체는 Date를 prototype으로 가지면서 var 이기 때문에 Object.prototype도 프로토타입으로 가집니다.
여기서 object와 같은 객체는 가장 상위에 존재하는 프로토타입이다. 

하나의 객체에 새로운 변수를 추가한다면 sql 에나 csv는 그 변수에 대한 칼럼이 새로 생기면서 다른 객체에는 NaN 값이 들어가지만!
자바스크립트는 그냥 하나의 객체에 정보가 하나만 추가된다.
